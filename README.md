# PROJEK ALIRA HUB @ SJK YUK KUNG

## Pengenalan / Introduction
Projek ALIRA HUB merupakan satu inisiatif di bawah Program Perantisan Kelayakan Profesional Nasional Bagi Peneraju Pendidikan (NPQEL) yang dijalankan di SJK Yuk Kung. Projek ini bertujuan untuk meningkatkan kecekapan, pengurusan, dan kecemerlangan penyampaian pendidikan digital serta akses maklumat secara berpusat.
